Display PDF documents in a responsive, configurable Elementor grid.

== Description ==

Add PDF documents through an Elementor repeater and display titles, descriptions, custom or generated thumbnails, and responsive layouts. The plugin includes extensive Elementor style controls and uses a local WordPress fallback icon when no PDF preview is available.

== Installation ==

1. Activate Elementor.
2. Activate Elementor PDF Grid Widget.
3. Edit a page with Elementor and add the PDF Grid widget.
